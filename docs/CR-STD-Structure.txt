ชื่อโครงการ : ศูนย์เชียงรายศึกษา

หน่วยงานผู้รับผิดชอบ : ฝ่ายทำนุบำรุงศิลปวัฒนธรรม คณะสังคมศาสตร์ มหาวิทยาลัยราชภัฏเชียงราย

ภาคีเครือข่ายความร่วมมือ
	1. สำนักศิลปะและวัฒนธรรม มหาวิทยาลัยราชภัฏเชียงราย
	2. สภาวัฒนธรรมจังหวัดเชียงราย
	3. สำนักงานวัฒนธรรมจังหวัดเชียงราย
	4. สำนักงานพัฒนาชุมชนจังหวัดเชียงราย
	5. คณะกรรมการพัฒนาสตรีจังหวัดเชียงราย
	6. สภาวัฒนธรรมอำเภอ 18 อำเภอในจังหวัดเชียงราย

หลักการและเหตุผล:
	จังหวัดเชียงรายเป็นหนึ่งในจังหวัดสำคัญทางภาคเหนือของประเทศไทยที่มีประวัติศาสตร์ยาวนานและวัฒนธรรมที่หลากหลาย ทั้งในด้านประเพณี ความเชื่อ ศิลปะการแสดง และภูมิปัญญาท้องถิ่น ซึ่งเป็นเอกลักษณ์ที่สะท้อนตัวตนและอัตลักษณ์ของชุมชนในพื้นที่ อย่างไรก็ตาม การเปลี่ยนแปลงทางสังคม เศรษฐกิจ และเทคโนโลยีที่รวดเร็วในยุคปัจจุบัน ได้ส่งผลกระทบต่อการสืบสานและการอนุรักษ์องค์ความรู้ที่เกี่ยวกับจังหวัดเชียงรายโดยเฉพาะอย่างยิ่งข้อมูลที่ยังไม่ได้รับการรวบรวม บันทึก จัดระบบ และเผยแพร่อย่างเป็นระบบ การจัดตั้งศูนย์เชียงรายศึกษาจึงมีความสำคัญในการรวบรวม จัดระบบข้อมูลองค์ความรู้ อนุรักษ์ และเผยแพร่ข้อมูลเหล่านี้ในรูปแบบที่เข้าถึงได้ง่าย เช่น ฐานข้อมูลดิจิทัล คลังความรู้ หรือนิทรรศการ อีกทั้งศูนย์เชียงรายศึกษาจะเป็นแหล่งสนับสนุนการวิจัยและการศึกษาเกี่ยวกับวัฒนธรรมท้องถิ่น โดยเฉพาะในประเด็นที่เกี่ยวข้องกับประวัติศาสตร์และภูมิปัญญาพื้นบ้าน อีกทั้งยังช่วยเสริมสร้างความรู้ ความเข้าใจ และความภาคภูมิใจในมรดกทางวัฒนธรรมของชุมชน
นอกจากนั้น ศูนย์เชียงรายศึกษายังมีบทบาทในการเชื่อมโยงนักวิชาการ หน่วยงานท้องถิ่น และชุมชน ให้เกิดการแลกเปลี่ยนองค์ความรู้และการทำงานร่วมกันเพื่อพัฒนาเชิงพื้นที่ การสร้างกิจกรรมที่เชื่อมโยงวัฒนธรรมเข้ากับการท่องเที่ยวเชิงสร้างสรรค์จะช่วยส่งเสริมเศรษฐกิจในท้องถิ่น นอกจากนี้ยังเป็นพื้นที่ในการแสดงออกถึงความหลากหลายทางวัฒนธรรม เช่น การแสดงศิลปะ งานหัตถกรรม และเทศกาลท้องถิ่น ในระยะยาว ศูนย์เชียงรายศึกษาจะเป็นกลไกสำคัญในการสืบสานวัฒนธรรมและองค์ความรู้ท้องถิ่น ควบคู่กับการสร้างโอกาสใหม่ๆ ในการเรียนรู้และพัฒนาให้จังหวัดเชียงรายสามารถรักษาอัตลักษณ์ที่เป็นเอกลักษณ์ได้อย่างยั่งยืน และยังเป็นตัวอย่างของการอนุรักษ์และพัฒนาที่สมดุลระหว่างอดีตและอนาคต
ดังนั้น การจัดตั้งศูนย์เชียงรายศึกษามีความสำคัญต่อการขับเคลื่อนงานด้านวิชาการในจังหวัดเชียงราย ได้แก่ 1.การอนุรักษ์วัฒนธรรมและภูมิปัญญาท้องถิ่น เนื่องด้วยเชียงรายมีประวัติศาสตร์และวัฒนธรรมที่โดดเด่น แต่ข้อมูลบางส่วนอาจเสี่ยงต่อการสูญหาย หากไม่มีการรวบรวมและจัดเก็บอย่างเป็นระบบ การสร้างศูนย์กลางจะช่วยปกป้องมรดกท้องถิ่นจากการถูกกลืนไปกับการเปลี่ยนแปลงทางสังคมและเศรษฐกิจ 2.การส่งเสริมความรู้และการศึกษา ศูนย์เชียงรายศึกษาจะเป็นแหล่งข้อมูลสำคัญสำหรับนักวิชาการ นักศึกษา และผู้สนใจทั่วไปที่ต้องการศึกษาวัฒนธรรมท้องถิ่นอย่างลึกซึ้ง เปิดโอกาสให้เกิดการพัฒนางานวิจัยและเผยแพร่องค์ความรู้ที่เกี่ยวข้องกับเชียงราย 3.การพัฒนาเศรษฐกิจและการท่องเที่ยวเชิงวัฒนธรรม ศูนย์เชียงรายศึกษาสามารถช่วยสนับสนุนการท่องเที่ยวเชิงวัฒนธรรม โดยดึงดูดนักท่องเที่ยวและผู้ที่สนใจให้มาสัมผัสประสบการณ์ที่เชื่อมโยงกับวัฒนธรรมและประเพณีของพื้นที่ ส่งเสริมการสร้างรายได้ให้กับชุมชนผ่านกิจกรรมที่เกี่ยวข้อง 4.การสร้างความภูมิใจในท้องถิ่น ที่ช่วยสร้างความตระหนักรู้และความภาคภูมิใจให้กับคนในพื้นที่ โดยเฉพาะคนรุ่นใหม่ที่อาจไม่คุ้นเคยกับรากเหง้าของวัฒนธรรมท้องถิ่น  และ 5.การเป็นศูนย์กลางความร่วมมือในพื้นที่ ศูนย์เชียงรายศึกษาจะเป็นจุดประสานงานระหว่างหน่วยงานภาครัฐ เอกชน นักวิจัย และชุมชน เพื่อพัฒนาและขับเคลื่อนกิจกรรมที่ส่งเสริมวัฒนธรรมและภูมิปัญญาท้องถิ่น ดังนั้นศูนย์เชียงรายศึกษาจะเป็นกลไกสำคัญในการอนุรักษ์และเผยแพร่วัฒนธรรมท้องถิ่น ขับเคลื่อนการศึกษาและการพัฒนาอย่างยั่งยืนในจังหวัดเชียงราย และช่วยเสริมสร้างคุณค่าให้กับชุมชนได้อย่างกว้างขวาง 

วัตถุประสงค์ 
1. เพื่อเป็นแหล่งข้อมูลและองค์ความรู้เกี่ยวกับประวัติศาสตร์ ประเพณีความเชื่อ ศิลปะการแสดง และภูมิปัญญาท้องถิ่นของจังหวัดเชียงราย: ซึ่งปัจจุบันข้อมูลสำคัญเหล่านี้กระจัดกระจายอยู่ในเอกสารหรือแหล่งข้อมูลต่างๆ ที่เข้าถึงได้ยาก ศูนย์เชียงรายศึกษาจะรวบรวมข้อมูลทั้งหมดไว้ในที่เดียวอย่างเป็นระบบ     ให้เป็นฐานข้อมูลที่ครอบคลุมและเป็นข้อมูลปัจจุบันที่สามรถเข้าถึงได้ง่าย เช่น ฐานข้อมูลดิจิทัลหรือคลังเอกสารออนไลน์
2. เพื่อรวบรวม อนุรักษ์ และเผยแพร่ข้อมูลเกี่ยวกับมรดกทางวัฒนธรรมของเชียงรายในรูปแบบต่างๆ: มีความจำเป็นในการอนุรักษ์ข้อมูลที่เกี่ยวข้องกับวัฒนธรรมท้องถิ่นก่อนที่จะสูญหายเนื่องจากการเปลี่ยนแปลงทางสังคมและเศรษฐกิจ การเผยแพร่ข้อมูลในรูปแบบต่างๆ เช่น นิทรรศการ สื่อสิ่งพิมพ์ และสื่อดิจิทัล จะช่วยให้ผู้คนในวงกว้างสามารถเข้าถึงข้อมูลเหล่านี้ได้สะดวก
3. เพื่อส่งเสริมการวิจัยและการศึกษาเกี่ยวกับวัฒนธรรมและภูมิปัญญาท้องถิ่นของเชียงราย: สร้างโอกาสให้นักวิชาการและนักศึกษาได้ทำวิจัยเชิงลึกในหัวข้อที่เกี่ยวข้อง สนับสนุนการพัฒนาวิทยานิพนธ์ ผลงานวิจัย และบทความวิชาการเพื่อเสริมสร้างองค์ความรู้ใหม่
4. เพื่อสนับสนุนกิจกรรมที่สร้างความตระหนักและความภาคภูมิใจในมรดกทางวัฒนธรรมของชุมชน: การจัดกิจกรรมหรือเวทีแลกเปลี่ยนความคิดเห็นระหว่างคนรุ่นใหม่และผู้รู้ท้องถิ่น สร้างพื้นที่ในการแสดงออกถึงคุณค่าของวัฒนธรรม เช่น การจัดแสดงศิลปะการแสดงและงานหัตถกรรม
5. เพื่อเป็นศูนย์กลางในการประสานงานและแลกเปลี่ยนความรู้ระหว่างนักวิชาการ หน่วยงานราชการ และชุมชนในพื้นที่: สนับสนุนเครือข่ายการทำงานร่วมกันระหว่างภาคส่วนต่างๆ เพิ่มโอกาสในการประสานความร่วมมือในด้านการจัดกิจกรรม การอนุรักษ์ และการพัฒนาท้องถิ่น

เป้าหมาย
1. จัดทำฐานข้อมูลดิจิทัลเกี่ยวกับประวัติศาสตร์ ประเพณีความเชื่อ ศิลปะการแสดง และภูมิปัญญาท้องถิ่นของจังหวัดเชียงราย
2. เผยแพร่ความรู้ในหัวข้อที่เกี่ยวข้องภายใต้แนวคิด 5 อัตลักษณ์นครเชียงราย ได้แก่ 1.ด้านประวัติศาสตร์  2.ด้านโบราณคดี  3.ด้านวัฒนธรรม ความเชื่อและประเพณี 4. ด้านศิลปะการแสดง 5. ด้านวิถีชีวิต อาชีพและความเป็นอยู่
3. สร้างเครือข่ายความร่วมมือชระหว่างหน่วยงานภาครัฐ ภาคเอกชน ภาคประชาสังคม องค์กรปกครองส่วนท้องถิ่น นักวิชาการ และผู้รู้ชุมชน ในการแลกเปลี่ยนเรียนรู้ด้านงานวิชาการที่เกี่ยวข้องกับจังหวัดเชียงราย
4. ผลิตผลงานวิชาการ งานวิจัย และสื่อที่เกี่ยวข้องกับวัฒนธรรมและภูมิปัญญาท้องถิ่นของจังหวัดเชียงรายในประเด็น 5 อัตลักษณ์นครเชียงราย

กิจกรรมหลัก
1. การรวบรวมข้อมูลและสร้างฐานข้อมูลดิจิทัลเกี่ยวกับประวัติศาสตร์และวัฒนธรรมเชียงราย
2. ผลิตหลักสูตรการจัดอบรมและสัมมนาในหัวข้อที่เกี่ยวข้อง
3. การผลิตผลงานวิชาการ และเผยแพร่หนังสือหรือสื่อมัลติมีเดียเกี่ยวกับวัฒนธรรมท้องถิ่นเชียงราย
4. การสนับสนุนงานวิจัยและโครงการอนุรักษ์เกี่ยวกับจังหวัดเชียงราย

สถานที่ดำเนินการ: คณะสังคมศาสตร์ มหาวิทยาลัยราชภัฏเชียงราย จังหวัดเชียงราย

งบประมาณ: ประมาณการงบประมาณขึ้นอยู่กับกิจกรรมและแหล่งสนับสนุน เช่น ทุนจากหน่วยงานรัฐ ท้องถิ่น หรือภาคเอกชน

ผลที่คาดว่าจะได้รับ
1. มีแหล่งข้อมูลที่ครอบคลุมเกี่ยวกับประวัติศาสตร์และวัฒนธรรมของจังหวัดเชียงราย
2. มีการอนุรักษ์และเผยแพร่ภูมิปัญญาท้องถิ่นอย่างเป็นระบบ
3. สร้างความร่วมมือระหว่างชุมชน นักวิชาการ และหน่วยงานท้องถิ่น และเป็นพื้นที่ในการแลกเปลี่ยนเรียนรู้ด้านงานวิชาการที่เกี่ยวข้องกับจังหวัดเชียงราย
4. เพิ่มความตระหนักในคุณค่าของมรดกทางวัฒนธรรมในระดับท้องถิ่นและระดับชาติ

ตัวชี้วัดความสำเร็จ
1. จำนวนข้อมูลหรือสื่อที่รวบรวมและเผยแพร่
2. จำนวนผู้เข้าร่วมกิจกรรมหรือนิทรรศการ
3. จำนวนผลงานวิจัยหรือสิ่งพิมพ์ที่ผลิตขึ้น
4. ความพึงพอใจของผู้เข้าร่วมกิจกรรมและผู้ใช้งานฐานข้อมูล
