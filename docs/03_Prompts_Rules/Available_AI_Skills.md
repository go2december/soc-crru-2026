# 🤖 สารบัญ AI Skills สำหรับโปรเจกต์นี้

นี่คือรายชื่อ "Skill พิเศษ" ที่ AI ในโปรเจกต์นี้ได้รับการติดตั้งไว้ (อยู่ในโฟลเดอร์ `.agent/skills/`)
คุณสามารถเรียกใช้ความสามารถพิเศษเหล่านี้ได้ง่ายๆ เพียงพิมพ์ `@` ตามด้วยชื่อสกิล หรือบอกให้ AI ดึงมาใช้ได้เลย

---

## 🎨 สายออกแบบและหน้าบ้าน (Frontend & Design)
*   **`[[ui-ux-pro-max]]`** - สกิลโหดสุดสำหรับคุม Theming, Layout, CSS, และเช็ค Checklist ความสวยงาม (คุณเพิ่งลองใช้ไป)
*   **`[[frontend-design]]`** - ช่วยคิดและออกแบบ Component (ปุ่ม, การ์ด, ฟอร์ม) รวมถึงแกะแบบจาก Wireframe
*   **`[[tailwind-patterns]]`** - ตัวตึงเรื่อง Tailwind v4 (และ v3) สำหรับทำ Responsive และแก้ปัญหา CSS ทับซ้อน
*   **`[[web-design-guidelines]]`** - ใช้ตอนอยากให้ช่วย "Audit" หน้าเว็บ ว่าผ่านเกณฑ์ Accessibility และ UX ดีหรือยัง

## ⚡ สายโค้ดดิ้งและสถาปัตยกรรม (Core Development)
*   **`[[clean-code]]`** - (บังคับใช้เสมอ) กฎการเขียนโค้ดให้สะอาด คลีน ไม่เยิ่นเย้อ อิงตามสไตล์มืออาชีพ
*   **`[[nextjs-react-expert]]`** - ผู้เชี่ยวชาญ Next.js App Router (โปรเจกต์นี้ใช้ v16.1.1) สำหรับจัดการ Server Components และ Optimization
*   **`[[api-patterns]]`** - การออกแบบ API, การตั้งชื่อ Endpoint, และจัดการ HTTP Status (เหมาะกับ NestJS หลังบ้าน)
*   **`[[database-design]]`** - กฎการทำ Schema ฐานข้อมูล และการใช้ Drizzle ORM
*   **`[[nodejs-best-practices]]`** - สำหรับคุมคุณภาพโค้ดฝั่ง Backend (NestJS)

## 🐛 สายแก้ปริศนาและทดสอบระบบ (Debug & QA)
*   **`[[systematic-debugging]]`** - สกิลนักสืบ 4 ระยะ! ใช้ตอนเวลาเจอ Error แดงเถือก แล้วหาทางแก้ไม่เจอ ให้สั่งเรียกสกิลนี้เลย
*   **`[[performance-profiling]]`** - ใช้ตอนเว็บโหลดช้า หรืออยากให้หาสาเหตุว่าทำไม Database Query ถึงนาน

## 🗺️ สายวางแผนและผู้ช่วย (Management & Workflow)
*   **`[[brainstorming]]`** - ใช้เมื่อคุณ "มีไอเดียแต่ยังไม่เคลียร์" สกิลนี้ AI จะสวมบทผู้สัมภาษณ์ ยิงคำถามเพื่อตบขอบเขตให้ชัดก่อนเขียนโค้ด
*   **`[[plan-writing]]`** - ใช้เขียนแผนงาน (Task Plan) แตกย่อยงานระบบใหญ่ให้เป็นชิ้นเล็กๆ 
*   **`[[seo-fundamentals]]`** - สกิลสำคัญสำหรับเป้าหมายปัจจุบัน! เอาไว้คุมเรื่อง OG Tags, Sitemap, และโครงสร้าง SEO ของเว็บศูนย์เชียงราย
