# Update Rules

## After Medium or Large Tasks

- Update `docs/WORKFLOW-project-status.md` when project status changes.
- Update the relevant `docs/PLAN-*.md` phase or checklist when feature status changes.
- Update the relevant `docs/WORKFLOW-*.md` file when operational behavior changes.
- Update `.windsurf/workflows/*.md` (and `.agent/workflows/*.md`) only when executable steps changed.

## New Operational Processes

- Document the process in `docs/WORKFLOW-*.md`.
- Add or update a matching executable workflow in `.windsurf/workflows/` (and `.agent/workflows/`).
