# Reproduce and Isolate

## Reproduce First

- Capture exact steps to reproduce.
- Record expected vs actual behavior.
- Note whether the issue is always reproducible or intermittent.

## Isolate the Scope

- What changed recently?
- Does it affect public pages, admin pages, backend APIs, or media handling?
- Can the problem be narrowed to one route, entity, API call, or component?
- Can the issue be reproduced with the smallest relevant flow?
