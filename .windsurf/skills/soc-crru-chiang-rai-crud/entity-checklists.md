# Entity Checklists

## Articles
- Verify admin list, create, edit, delete, `by-id`, and public slug detail page.
- Verify slug, tags, publish status, metadata, and sitemap/detail consumers.

## Artifacts
- Verify admin list, create, edit, delete behavior.
- Verify category mapping, archive filters, and artifact detail compatibility.

## Activities
- Verify admin list, create, edit, delete, `by-id`, and public slug detail page.
- Verify type enum, pagination, event fields, and image cleanup behavior.

## Learning Sites
- Verify admin list, create, edit, delete, `by-id`, and public slug detail page.
- Verify thumbnail, media gallery, content images, tags, and removed-image cleanup behavior.

## Staff
- Verify admin list, import, delete, and grouped display behavior.
- Verify faculty import compatibility and role/group mapping.

## Settings
- Verify settings form load/save behavior.
- Verify homepage hero image, title, subtitle, and dependent homepage sections.
