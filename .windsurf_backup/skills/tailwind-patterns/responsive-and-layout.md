# Responsive and Layout

## Layout Rules

- Write mobile-first styles first, then layer larger breakpoints deliberately.
- Prefer reusable layout patterns over ad hoc one-off utility combinations.
- Use grid/flex based on content structure and maintenance clarity.

## Review Focus

- Public pages: landing sections, content grids, detail layouts
- Admin pages: forms, summary cards, table wrappers, modal responsiveness
- Responsive polish across mobile, tablet, and desktop
