# Usage Notes

## Upstream Reference

This local skill is adapted from the public `nextlevelbuilder/ui-ux-pro-max-skill` concept and tailored to the SOC-CRRU repository.

## Recommended Prompting Style

Use prompts such as:

- "Improve the public landing page UX for this faculty route using DaisyUI only"
- "Refine this admin CRUD page using shadcn/ui and keep the existing auth flow"
- "Make this Chiang Rai public detail page more polished without changing route structure"

## Integration Note

If you later want the full upstream package installed directly, the upstream project indicates support for Windsurf installation via its own CLI. This local wrapper exists so the repository can use the same design intent immediately without depending on an external installer.
