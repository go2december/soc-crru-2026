# Structured Data Checks

## JSON-LD Rules

- Verify JSON-LD remains aligned with the page content and entity type.
- If detail/list page semantics change, revisit structured data output.
- Check non-detail public pages as part of broader SEO audit or release readiness.

## Validation Focus

- Detail pages for articles, activities, learning-sites, and archive entities
- Public static/list pages affected by content-model or route-structure changes
